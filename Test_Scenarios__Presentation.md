## 🧪 Test Scenario Documentation

This file contains manual test scenarios created to demonstrate my software testing knowledge and QA thinking. The scenarios cover testing for different products and applications such as hardware devices, web applications, and mobile applications.

Each scenario is written with TS numbering and focuses on verifying functionality, usability, and reliability from a tester’s perspective. This project is part of my QA learning portfolio for interview preparation.

🎯 Purpose

These test scenarios are created for:

QA learning and practice

Portfolio demonstration for QA interviews

Understanding how to design structured test scenarios for different systems
